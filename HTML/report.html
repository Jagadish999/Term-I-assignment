Designing of Home page

Now the most difficult part for me was in designing the home page and took most of my time because I had no idea about portfolio websites. After getting idea by the portfolio website of famous web designer Jack kjeznach and some other portfolio website I decided a layout for my website.

The navigation buttons are created as 3D buttons with multiple shadows. This concept was given by a you tuber Ryan in his video “3D Text Effect in CSS/Ryan Web Tech” and with some use of his idea in code line 42 to 53, 166 to 173, 230 to 240, 282 to 292 the element looks like 3d object. Another code that I searched was making text filling transparent and giving an outline in text which is used in heading in home page as well as in text logo. In code line number 137 and 138, 143 to 153, 228 and 229 this code was used which is available on website: https://css-tricks.com/adding-stroke-to-web-text/. This codes are not completely copied but still same idea exist here. Finally after this I decided to change some background color and some spacing and finalized my layout. The screenshot of validation of home page is also included.

Talking about the responsive view in mobile devices, it has some changes in its layout. The gaping and margins are slightly decreased and a hamburger menu appears which can be opened and closed by clicking it. Similarly, the navigation menu is also redesigned where each menu is displayed vertically after a click on hamburger menu.

Designing of Contact Page

The contact page contains a navigation menu as similar to every page. This page contains a form and my contact information. I have chosen simple neon outline form and a card containing my information. Animation in this page is also very similar to other pages. The idea of form is from neon text effect which is shown in website: https://css-tricks.com/how-to-create-neon-text-with-css/ . The difficulty was creating shadow inside the border on input elements and I found the way on same website: https://css-tricks.com/snippets/css/css-box-shadow/ which is used in code line 497 in CSS. This page also have some changes in mobile view all the main elements are displayed vertically and same hamburger menu functioning as other pages. On submitting the form, mail box appears as explained in assignment. This is idea was in https://www.w3schools.com/tags/tag_address.asp, which is used in code line 39 of html page.
